### Installation
```lua
-- Using packer
use 'MysteryMage/tavern.nvim',

-- Using lazy
'MysteryMage/tavern.nvim',

```

### Transparency
```lua
-- Somewhere in your config before using the colorscheme
vim.g.tavern_transparent = true
```
